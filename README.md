# Java Pathfinding Engine (BE Graphes)

## Description
Ce projet est une application Java permettant de calculer des itinéraires routiers (plus court chemin) sur des cartes réelles (ex: Toulouse, France). Il compare les performances de différents algorithmes de théorie des graphes.

## Fonctionnalités
* **Algorithmes :** Implémentation de **Dijkstra** et **A*** (A-Star).
* **Données :** Manipulation de grands graphes représentant des réseaux routiers.
* **Visualisation :** Interface graphique pour visualiser les nœuds, les arcs et le chemin calculé.

## Stack Technique
* **Langage :** Java
* **Architecture :** Modèle-Vue-Contrôleur (séparation `core`, `gui`, `model`).
